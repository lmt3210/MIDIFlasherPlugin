Welcome

MIDIFlasher is a music device audio unit that displays up to 99 different colors based on MIDI input. One practical use is as a visual metronome.

Usage

Insert MIDIThru into a DAW instrument track. MIDI note on messages will determine the color displayed per the table below. Note off messages and note numbers not in the table will display the background color.

Note names in the table consider middle C to be MIDI note 60 or C4. Korg considers C4 to be middle C; Roland and Yamaha consider C3 to be middle C. Many DAWs allow the user to select C3, C4, or sometimes C5 to be middle C. Please consider this when attempting to select a specific color via MIDI.

All MIDI traffic except system messages (system exclusive, sequencer start, MIDI clock, active sensing, etc.) are passed through MIDIFlasher unchanged.
